Point cloud processing 2018

a 1-day seminar on point cloud acquisition, data structures, classification, segmentation and applications
March 13th, 2018 · Delft, the Netherlands

Point cloud processing 2018 is a 1-day seminar on the latest developments in point cloud processing. It will be held on March 13 2018 at the Delft University of Technology and is organised at the occasion of the PhD defense of Ravi Peters. One day later by defending his dissertation titled "Geographical point cloud modelling with the 3D medial axis transform", which is part of the NWO-funded 3DSM project.

This event is hosted by the 3D geoinformation group and supported by EuroSDR and the VOLTA project.

The objective of this seminar is share knowledge about exciting developments in point cloud processing. Topics of interest are:

  • acquisition & visualisation
  • data structures & management,
  • segmentation & classification,
  • object detection & reconstruction,
  • applications & use-cases from practice.
